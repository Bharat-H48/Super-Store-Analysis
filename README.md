# SuperStore Sales Analysis

1. Project Overview
- Business Intelligence analysis of SuperStore sales data using **Python** and **Power BI**.  
- The goal is to uncover sales trends, profit margins, and growth opportunities to improve company performance.

2. Key Steps
- Cleaned and prepared dataset (5,901 rows, 20 features).
- Converted dates for accurate time-series analysis.
- Imputed missing values in Returns column.
- Built interactive dashboards in Power BI.

3. Insights
- Total Sales: $10.1M | Profit: $970K | Margin: 9.6%
- West region drives 33.4% of revenue.
- Office Supplies are most profitable; Furniture shows weak margins.
- Q4 sales spike ~25% above average.

4. Recommendations
- Optimize costs in Furniture category.
- Invest in Central region to grow revenue.
- Plan inventory 90 days ahead for Q4 surge.
